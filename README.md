# 7Q-Station-Manager: logging modules

[![ci](https://github.com/7Q-Station-Manager/logging/actions/workflows/ci.yml/badge.svg)](https://github.com/7Q-Station-Manager/logging/actions/workflows/ci.yml)

This module provides logging functionality for the 7Q-Station-Manager project. It includes a custom logger that is
used throughout all the applications to log messages at different levels (debug, info, warning, error, critical).
